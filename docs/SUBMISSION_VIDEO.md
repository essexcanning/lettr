# 1-min submission video — shotlist

Required by the hackathon submission form. Must show only what was built today.
Record with QuickTime Screen Recording (Cmd-Shift-5 on Mac, "Record Selected Portion").

Total length: **60 seconds**.

## Shotlist

| Sec | What's on screen | Voiceover (Stephen) |
|---|---|---|
| 0–5 | Lettr README at the top of the repo | "Lettr — a voice-first AI letting agent for London." |
| 5–15 | LiveKit playground; you talk to Lettr giving the spec | "I tell it what I want…" *[live audio of the exchange]* |
| 15–25 | Dashboard reputation table | "…Lettr searches MongoDB Atlas Vector Search, contacts the top letting agent, and tracks every interaction in a public reputation graph." |
| 25–40 | Timeline event with voicemail audio. Press play. | "When the agent ghosts, Lettr picks up the phone." *[audio of voicemail playing]* |
| 40–50 | Atlas Collections view showing `letting_agents`, `lettr_tasks`, `langgraph_checkpoints` | "Built on MongoDB Atlas, LangGraph with the MongoDB Checkpointer, LiveKit, ElevenLabs, and Fireworks." |
| 50–60 | Dashboard final state showing chronic_ghoster flags + viewing booked | "Theme: Prolonged Coordination. Built today. Stephen Canning and Lea Rattei." |

## Recording tips

- Record at 1080p, 30fps minimum.
- Get the audio levels right — voicemail moment must be audible.
- Name the file `lettr-submission.mp4`.
- Upload to YouTube (Unlisted) or Loom — paste the link into the submission form.
- Aim to record by **16:30** so there's time for a re-record if it flops.
